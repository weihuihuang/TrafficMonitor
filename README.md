# TrafficMonitor
简单的项目网络监测工具,记录自悬浮窗口打开之后，app的流量总消耗、网络状况、每秒消耗流量  
用法：在需要开启的地方调用TrafficMonitoringManager.startPollingTask(getContext(), NET_POLLING_PRIOD_MILLIS);
NET_POLLING_PRIOD_MILLIS是监测轮询周期
